Модуль 1 - Домашнє завдання

Зверстати сторінку на основі макета homework-01.psd

  1. Перекласти макет сторінки в HTML-розмітку, без CSS оформлення.
  2. Зображення наріжте з макета.
  3. Підготовлені зображення розмістіть в папці img в папці проекту.
  4. Весь текстовий контент взяти з макета.
  5. Готове домашнє завдання буде виглядати як на скріншоті нижче.
