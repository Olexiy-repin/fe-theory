Модуль 2 - Домашнє завдання

  1. Створити обліковий запис на GitHub
  2. Розібратися з основами Github по туторіали githowto.com. У туторіали також можна знайти інструкції по установці Гіта на вашу операційну систему.
  3. Зробити репозиторій на GitHub, назвати його goit-fe-course
  4. Зробити клон створеного сховища на свій комп'ютер
  5. Додати домашню роботу номер 1 в папку склонірованого сховища. Папки для домашніх робіт називати module-n (де n це номер модуля, наприклад module-01)
  6. Додати всі зміни в git і зробити Комміт
  7. Запушити зміни в віддалений репозиторій
  8. Використовуючи GitHub Pages, налаштувати репозиторій на роздачу статичних файлів
