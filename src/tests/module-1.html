Модуль 1 - Тесты

  1. Что не является HTML-тегом?

    1. <c></c>
    2. <a></a>
    3. <b></b>
    4. <i></i>

    Правильный ответ: 1

  2. Что не является атрибутом тега a?

    1. alt
    2. href
    3. target
    4. title

    Правильный ответ: 1

  3. Какой из этих тегов не строчный?

    1. <p></p>
    2. <a></a>
    3. <b></b>
    4. <i></i>

    Правильный ответ: 1

  4. Какой из примеров кода пройдет валидацию?

    1. Ни один из примеров не пройдет валидацию
    2. <b>Hello <b>World!</b></b>
    3. <img src = "" alt = "">
    4. <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ducimus ratione, omnis eveniet deleniti, error earum blanditiis ullam doloribus repudiandae esse cumque corrupti perspiciatis sit rerum, sapiente nulla modi placeat aut?</p>

    Правильный ответ: 1

  5. Выберите валидный HTML комментрий:

    1. <!-- this is a comment -->
    2. // this is a comment
    3. /* this is a comment */
    4. # this is a comment

    Правильный ответ: 1

  6. Какой тег может быть вложен непосредственно (первая вложенность) внутрь списков (ol и ul)?

    1. только li
    2. любой
    3. только блочные теги
    4. только строчные

    Правильный ответ: 1

  </p>

  8. Укажите тег, размер шрифта которого по-умолчанию больше других.

    1. h4
    2. h6
    3. p
    4. span

    Правильный ответ: 1

  9. Выберите верное описание поведения потока документа для европейских стран.

    1. Сверху вниз, слева направо
    2. Сверху вниз, справа налево
    3. Сверху вниз, слева направо и смена направления при переходе на новую строку
    4. Снизу вверх, справа налево

  10. Блочный элемент от строчного отличается тем что...

    1. Занимает всю доступную ширину родителя
    2. Ширина определяется контентом
    3. Внутри нельзя размещать другой блочный элемент
    4. Не обязан начинаться с новой строки

    Правильный ответ: 1
