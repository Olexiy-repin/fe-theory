Модуль 5 - Тесты

  1. Свойство position: relative сместит соседний элемент при значении свойства...

    1. margin: 50px;
    2. top: 50px;
    3. left: 50px;
    4. bottom: 50px;
    5. right: 50px;

    Правильный ответ: 1

  2. Свойство position: relative может перекрыть изображение соседнего элемента при значении свойства...

    1. top: 50px;
    2. padding: 50px;
    3. margin: 50px;

    Правильный ответ: 1

  3. Свойство position: absolute начнет отсчет для своего положения от...

    1. родительского элемента со свойством position: relative;
    2. родительского элемента со свойством display: block;
    3. родительского элемента со свойством display: float;
    4. родительского элемента со свойством clear: both;
    5. родительского элемента со свойством position: static;

    Правильный ответ: 1

  4. Свойство position: fixed применяют для...

    1. создания блока закрепленного во viewport
    2. для создания выпадающего меню блока навигации
    3. навигации внутри заданного блока страницы
    4. навигации между страницами

    Правильный ответ: 1

  5. Свойство position: sticky применяют для...

    1. фиксации элемента внутри заданного блока страницы
    2. фиксации элемента выпадающего меню
    3. фиксации элемента на странице
    4. фиксации элемента при переходе между страницами

    Правильный ответ: 1

  6. Свойство z-index работает для...

    1. только для элементов с position отличным от static
    2. любого элемента на странице
    3. любого элемента в потоке
    4. любого элемента вне потока
    5. только для элементов с position: absolute

    Правильный ответ: 1

  7. Верстка выпадающего меню использует правило для появления скрытого меню в виде...

    1. .menu-item:hover > .submenu { display: block; }
    2. menu-item:hover > .submenu { display: none; }
    3. submenu-item:hover > .menu { display: block; }
    4. .menu:hover > .submenu { display: block; }

    Правильный ответ: 1

  8. Семантическая верстка нужна для...

    1. выделения контента по его содержанию
    2. ускорения загрузки страниц
    3. улучшения внешнего вида страниц
    4. выделения контента по его внешнему виду
    5. для удобства верстки

    Правильный ответ: 1
