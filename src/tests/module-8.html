Модуль 8 - Тесты

  1. Адаптивный сайт это...

    1. Сайт который имеет всего несколько вариантов отображения в зависимости от ширины экрана или типа устройства.
    2. Сайт который отображается на любом устройстве одинаково.
    3. Сайт который плавно изменяет свой вид при изменении ширины вьюпорта.
    4. Сайт который отображается так как того хочет пользователь

    Правильный ответ: 1

  2. Отзывчивый сайт это...

    1. Сайт который имеет всего несколько вариантов отображения в зависимости от ширины экрана или типа устройства и плавно изменяющий свой вид при изменении ширины вьюпорта.
    2. Сайт который имеет всего несколько вариантов отображения в зависимости от ширины экрана или типа устройства, без плавных переходов.
    3. Сайт сделаный для отображения только на телефонах
    4. Сайт который отображается на любом устройстве одинаково.

    Правильный ответ: 1

  3. По какой формуле рассчитываются пропорции ширины блока при респонсив верстке?

    1. ширина блока / ширина родителя * 100
    2. ширина родителя - ширина блока / 100
    3. ширина родителя * ширина блока / 100
    4. ширина родителя / ширина блока * 100

    Правильный ответ: 1

  4. Зачем используется метатег viewport?

    1. Для того, чтобы сообщить браузеру каким образом нужно контролировать размеры и масштаб страницы
    2. Для того, чтобы страницу можно было роастягивать.
    3. Для того, чтобы работали медиаправила.
    4. Для того, чтобы веб станица хорошо выглядела на телефонах

    Правильный ответ: 1

  5. Для чего задаются min-width и max-width блокам-контейнерам?

    1. Для того чтобы контейнер подстраивался под текущую ширину окна браузера, но не растягивался шире чем максимальная ширина, и не сжимался уже чем минимальная ширина
    2. Чтобы веб страницу можно было растягивать.
    3. Чтобы на странице заработали медиаправила
    4. Для того чтобы контейнер подстраивался под текущую ширину окна браузера

    Правильный ответ: 1

  6. Какое css правило сделает картинки на странице responsive?

    1. img { display: block; max-width: 100%; height: auto; }
    2. img { display: block; width: 100%; }
    3. img { display: block; width: 100%; height: 100%; }
    4. img { display: block; max-width: 100%; height: 100%; }

    Правильный ответ: 1

  7. Что описывает следующее медиаправило?
    @media (min-width: 768px) and (max-width: 1200px) { body { background-color: red; } }:

    1. Цвет фона body изменится на красный при ширине экрана от 768px до 1200px
    2. Цвет фона body изменится на красный при ширине экрана от 0px до 768px
    3. Цвет фона body изменится на красный при ширине экрана от 768px и шире
    4. Цвет фона body изменится на красный при ширине экрана от 0px до 1200px

    Правильный ответ: 1

  8. Сколько медиаправил можно применить к css правилу?

    1. любое количество
    2. 1
    3. 5
    4. 10

    Правильный ответ: 1

  9. Что такое source?

    1. Тег, используется для указания пары медиаправило-изображение для picture
    2. Аттрибут тега picture, используется для указания источника изображения
    3. Тег, используется для указания дефолтного изображения для picture

    Правильный ответ: 1

  10. При mobile-first подходе используются только min-width медиаправила?

    1. Нет, иногда можно элегантно применить max-width даже при mobile-first подходе
    2. Да, это стандарт, применять max-width категорически нельзя

    Правильный ответ: 1
