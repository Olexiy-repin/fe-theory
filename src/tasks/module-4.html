Модуль 4 - Завдання

Завдання 1

Створити список з шести елементів. За допомогою псевдокласу hover задати стилі елементам меню при наведенні курсору. Для створення горизонтального меню використовувати float.

Завдання 2

Створити список з дев'яти елементів. Для елементів списку, за допомогою псевдокласу nth-child, зробити 3 послідовних фону #88292F, #5DB7DE і #0D5D56. Колір шрифту повинен бути #0E1116.

Завдання 3

Створити список з дев'яти елементів. З допомогою псевдоелемента ::before поставити перед кожним посиланням символ стрілки.

Завдання 4

Створити список з 5-ти елементів. Використовуючи CSS лічильники і псевдоелемент before для li, пронумерувати елементи списку римськими цифрами, CSS оформлення на свій смак. Псевдоелемент повинен бути квадратом з вертикально і горизонтально центрованим текстом.
